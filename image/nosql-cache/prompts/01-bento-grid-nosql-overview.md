Title: NoSQL 是什么？一张图看懂

Hero cell - 中心大字:
「NoSQL = Not Only SQL（不只是表格 SQL）」
通俗说：不把所有数据都塞进一张严格表格里

Cell 1 - 关系型数据库（SQL）:
- 数据像 Excel 多张关联表
- 行和列结构固定
- 擅长：转账、订单、强一致
- 代表：PostgreSQL、MySQL

Cell 2 - 键值型（Key-Value）:
- 像超大字典：键 → 值
- 读写极快，结构简单
- 擅长：缓存、会话、计数器
- 代表：Redis

Cell 3 - 文档型（Document）:
- 一条记录 = 一份 JSON 文档
- 字段可以灵活增减
- 擅长：用户资料、商品详情
- 代表：MongoDB

Cell 4 - 列族 / 宽列（了解即可）:
- 按列族存储海量数据
- 代表：Cassandra、HBase

Cell 5 - 图数据库（了解即可）:
- 存「点」和「边」的关系
- 代表：Neo4j

Bottom takeaway:
「没有银弹：SQL 管账，Redis 管快，文档库管灵活」
