Title: Apache Tika 是什么

CENTER hub「Apache Tika」:
- 内容检测 + 文本抽取
- Java 生态老牌瑞士军刀

SPOKES:
- MIME 类型识别（魔数 + 启发式）
- 从 PDF/DOC/邮件等抽纯文本
- 可嵌入 Java 应用
- 可独立 HTTP Server（/tika）
- Python 通过 tika-python 或 HTTP 调用
