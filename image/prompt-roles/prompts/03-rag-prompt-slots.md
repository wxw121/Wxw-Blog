Title: RAG 提示词三槽位

Top layer (stable): System — 行为契约：只根据资料、不足则拒答、引用编号

Middle layer (per turn): User — 【资料】chunk 原文 + 元数据编号

Bottom layer (per turn): User 同条或后续 — 【问题】用户真正要问的

Side note: Assistant = 生成答案（不是把资料假装成自己说过的话）

Bottom coral: 指令 / 证据 / 问题分开放，降低糊成一团与幻觉乱发挥

Footer: System/User/Assistant 完全指南 · §6
