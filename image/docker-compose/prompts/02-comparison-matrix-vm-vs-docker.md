Title: 虚拟机 vs Docker 容器——差别在哪？

Rows comparing 虚拟机 Virtual Machine vs Docker 容器:

启动速度: VM 分钟级 vs 容器 秒级
资源占用: VM 几 GB 起 vs 容器 MB～百 MB
隔离级别: VM 完整操作系统 vs 容器 进程级隔离
系统内核: VM 各自内核 vs 容器 共享宿主机内核
典型用途: VM 跑不同 OS vs 容器 打包应用环境
开发体验: VM 重 vs 容器 一条 docker run

Bottom highlight: 开发部署常用 Docker，不是替代 VM 而是分工
